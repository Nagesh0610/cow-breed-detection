# 🐮 Cow Breed detection App

A Deep Learning web application that classifies cow breeds using EfficientNet-B0.

## 🚀 Features
- Upload cow images (JPG/PNG)
- Predict cow breed
- Display breed information
- Simple Streamlit UI

## 🧠 Model Details
- Model: EfficientNet-B0
- Framework: PyTorch
- Input Size: 224x224
- Saved Model: best_model.pth

## 🛠️ Technologies
- Python
- Streamlit
- PyTorch
- JSON

## 📦 Installation

```bash
pip install -r requirements.txt
streamlit run app.py
```
